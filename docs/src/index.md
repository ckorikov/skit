# Overview

`skit` (Syllabus Kit) is a tool for building study plans and learning
programs. It works with a curriculum: topics, the dependencies between
them, modules, and the study plans built from them.

## Core

- [Concepts](core/concepts.md) — the entities `skit` operates on.
- [The curriculum graph](core/graph.md) — nodes, edges, and weights.
- [Identifiers and Attributes](core/attributes.md) — how entity
  properties are stored.
- [Syllabus construction](core/syllabus.md) — modules, the build
  function, and syllabus structure.

## Extensions

- [Literature](extensions/literature.md) — topic sources and
  recommended reading lists.
- [Assessment](extensions/assessment.md) — grading, the score log,
  spaced repetition.
